使用活动监视器检测你的Mac电脑

Using Activity Monitor to Easily Troubleshoot Your Mac

活动监视器很像 Windows 中的任务管理器。它显示了当前在 Mac 上运行的所有应用程序的列表，以及他们正在占用的系统资源的大小。如果你以前使用过活动监视器，那可能是用它来强制退出某些应用程序。但是，它还有更强大的功能，并且可以显示关于管理系统资源甚至升级系统的重要信息。

要启动活动监视器，请在 Spotlight 或 Launchpad 中输入应用程序的名称。你也可以在“应用程序->实用工具->活动监视器”找到应用程序。

熟悉活动监视器

在默认视图中，活动监视器显示了所有当前正在运行的应用程序列表以及它们占用的 CPU 大小。它不只是显示用户应用程序，你还会看到很多系统进程和守护进程。

在窗口顶部，你会看到五个选项卡：CPU，内存，能耗，磁盘和网络。这些选项卡显示每个类别的资源使用数据。

在窗口底部，你将看到当前资源分配和一些高级统计信息的图形表示。对于CPU，你会看到系统进程和用户进程之间的分割。当你更改标签时，此图表将会更改以反映相关信息。

强制退出应用程序

活动监视器最常见的用途可能是强制退出应用程序。由于活动监视器可以让你看到当前正在运行的所有程序，因此可以快速识别已挂起或崩溃的应用程序。为了便于识别，挂起的应用程序名称将变为红色，文本“不响应”将显示在括号中。 这让你知道你需要强制退出该应用程序。

点击应用程序名称并点击“强制退出”按钮就可以强制退出该程序。

发现失控的应用

与任何表视图一样，你可以单击列的名称以按该列的标准进行排序。例如，单击“% CPU”将按其使用的 CPU 容量的百分比来排序应用程序。

这很快就可以找到资源占用最多的应用程序。当你发现电脑风扇突然开始转动时，你就可以查看这个排序列表，找出可能的罪魁祸首。

电脑是否需要加内存了

如果你要购买新的 Mac，则通过检查活动监视器的内存选项卡可以帮助你确定是否需要比目前更多的内存。
点击顶部的“内存”选项卡，并查看窗口底部的图形。

找到内存压力图，如果那是黄色或红色，那意味着你的系统内存不足。你还需要检查中间列底部的“已使用的交换”字段。如果这是一个很大的数字，这意味着你的计算机经常使用慢速硬盘存储器来填补内存不足。这大大减慢了系统的速度，这可能意味着你没有足够的内存来支持当前的工作。如果你看到更像下面的截图，那么你的电脑状态很好。

释放内存

当然，如果你不能增加更多的内存条，你也可以从活动监视器退出内存占用过高的应用程序。点击“内存”列按最内存密集的应用程序进行排序，然后开始关闭它们。

查找应用使用的网络带宽

单击顶部的“网络”选项卡可以显示一个表格，显示每个应用程序正在进行多少通信。你可以按“已发送字节”或“已接收字节”进行排序，以查看哪些应用程序是最活跃的。
