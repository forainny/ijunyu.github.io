Missing iOS Distribution signing identity for

发表于 2016-02-18 | 更新于 2017-06-04 | 分类于 iOS | |
  1. 下载 https://developer.apple.com/certificationauthority/AppleWWDRCA.cer
  2. 双击安装到钥匙串。
  3. 钥匙串 -> 显示 -> 显示已过期的证书。
  4. 钥匙串区域选择系统标签。
  5. 删除 Apple Worldwide Developer Relations Certificate Authority certificates。
